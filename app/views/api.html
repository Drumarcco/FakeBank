Realizar una transacción

Nuestro servidor utiliza JSON para comunicar datos, la estructura que requerimos para realizar una transacción es la siguiente:

Envia un JSON a través de una petición HTTP POST con el formato anterior a la dirección
{{basePath}}api/Transaction
y si se logra completar la transacción, el servidor te responderá con HTTP status Code 200, de lo contrario, te responderá con HTTP Status Code 400 (Bad Request).
